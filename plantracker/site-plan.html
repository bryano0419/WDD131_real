Daily Dev Tips Site Plan

Bryan Osorio

WDD 131-01

Overview

Purpose

This website helps beginner web developers stay consistent by offering quick and actionable daily coding tips. It provides brief learning content, code snippets, and motivation to improve daily coding habits.

Audience

This site is for students, self-taught developers, and junior programmers looking to boost their development skills one day at a time. It’s aimed at anyone wanting to learn web development in bite-sized chunks.

Dynamic elements

JavaScript will be used to display a random daily development tip from an array of objects. The site will feature filtering by category, interactive buttons, conditional messages, and collapsible sections for extra content. Users can sort, expand, or hide tips.

Content

Home page

The home page will feature a random daily tip, a welcome message, category filters (HTML, CSS, JS), and a motivational quote. It includes a hero image and a brief description of the site’s purpose.

Tips Archive Page

This page will list all previous tips categorized by topic. Tips will be stored in JavaScript arrays and dynamically displayed. Each will include a title, brief summary, and expandable detail.

Wireframes

Home

The homepage features a tip of the day card, filters for categories, and a short intro section. The motivational quote will be in the footer.

Tips Archive

The archive page shows tip cards in a grid layout with filtering buttons. Each card expands to show more details and can be sorted by category or date.
